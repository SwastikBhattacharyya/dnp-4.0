Bootstrap Directory Structure

Understanding the Bootstrap file organization

CSS Directory
  • bootstrap.css - Unminified CSS
  • bootstrap.min.css - Minified CSS
  • bootstrap.css.map - Source map file
JS Directory
  • bootstrap.bundle.js - Unminified JS with Popper.js
  • bootstrap.bundle.min.js - Minified JS with Popper.js
  • bootstrap.js - Core JavaScript
Icons (Optional)

Bootstrap Icons are available as a separate package and include:

  • SVG-based icons
  • Icon fonts
  • Multiple formats
